GenAI Configuration

By default, the assistant runs on a locally-compiled Semantic Rule Parser which handles standard data aggregations, filters, and rankings without any configuration.

To enable advanced, open-ended natural language reasoning, provide a Google Gemini API Key. The backend will switch to the official google-genai SDK and use gemini-2.5-flash to write Pandas queries on the fly.

The key is processed locally on your server and is not sent to any third party other than Google's Gemini endpoint.
